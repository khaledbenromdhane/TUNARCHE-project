Artvista - Symfony Web Application
🎨 About the Project

Artvista is a modern digital art platform developed as an academic integrated project using Symfony for the web application and JavaFX for the desktop application.

The platform combines art, artificial intelligence, online learning, event management, blogging, and e-commerce features into one complete ecosystem designed for artists, learners, and creative communities.

🚀 Features
🔐 Authentication & Security
User authentication and account management
Face recognition login system
Secure access for users and administrators
📚 Formation Management
Add and manage formations through the back office
AI-generated evaluation questions
QR code generation for formations
Online evaluations and grading system
Automatic certificate generation
Share certificates on LinkedIn and WhatsApp
📝 Blog & Community System
Create and publish posts
AI-generated titles and descriptions from uploaded images
Comment and reaction system
Emoji integration for comments
Bad words detection and moderation
🎟️ Event Management
Add and manage events
AI-generated event descriptions
Event participation system
Personalized QR code tickets
Stripe payment integration
Cash and online payment options
QR code verification by event moderators
🖼️ Gallery & Artwork Marketplace
Create and organize galleries
Add artworks with descriptions and pricing
Artwork statistics and analytics
Online artwork purchasing system
Secure Stripe payment integration

🎯 Project Objective

The objective of Artvista is to create an intelligent and interactive digital platform where users can learn, create, share, attend events, and purchase artworks in a secure and modern environment.

📬 Team Project

This project was developed as part of an academic integrated project combining web and desktop technologies with teamwork and collaborative development.
